## 🎨 Customize in 3 Steps

### Step 1: Change Your Business Info
Open [index.html](index.html) and replace:
- "Dolce" → Your shop name
- "@dolcesandyhook" → Your Instagram
- Phone, address, hours → Your details

### Step 2: Add Your Colors
Open [styles.css](styles.css), find `:root` and change:
```css
--color-accent-red: #E84855;    /* Your color */
--color-accent-blue: #4A90E2;   /* Your color */
```

### Step 3: Add Your Photos
- Save photos in the `images/` folder
- In [index.html](index.html), replace Unsplash URLs with:
  ```html
  <img src="images/your-photo.jpg" alt="Description">
  ```

**Done!** Refresh your browser to see changes.

## 📱 Test on Your Phone

1. Start a local server:
   ```bash
   python -m http.server 8000
   ```

2. Find your computer's IP:
   - Windows: Run `ipconfig`
   - Mac/Linux: Run `ifconfig`

3. On your phone browser:
   ```
   http://YOUR_IP:8000
   ```

## 🆘 Need Help?

### Common Questions:

**Q: How do I open the website?**
A: Double-click `index.html`

**Q: How do I change colors?**
A: Edit `:root` section in `styles.css`

**Q: How do I add my photos?**
A: Place them in `images/` folder and update `index.html`

**Q: Site looks broken?**
A: Clear cache (Ctrl+Shift+R) and ensure all files are in the same folder

**Q: How do I deploy for free?**
A: Use GitHub Pages, Netlify, or Vercel (see [DEPLOYMENT.md](DEPLOYMENT.md))
